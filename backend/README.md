# SourceBase Backend (Swift)

Swift (SwiftPM/Vapor) SourceBase backend, preserved here after the standalone
`medasi-sourcebase` GitHub repo was deleted. 

Status (2026-06-15): NOT a live service. `sourcebase.medasi.com.tr` currently
serves only an nginx placeholder page. The real backend used by the apps is the
shared Supabase project (medasi.com.tr) + its `sourcebase` edge function. This
Swift package is kept for reference / future use.

`REPOMIX.md` is a flattened single-file dump of the source for quick reading.
